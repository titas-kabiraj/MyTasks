# Tasque — Aesthetic To-Do App

Velora is a clean, minimal, and visually soothing to-do application designed to help you stay organized while enjoying a calm, elegant user experience. Built using pure HTML, CSS, and JavaScript, it combines functionality with a soft, modern design.

---

## Features

### Task Management

* Add new tasks with a single click or by pressing **Enter**
* Assign categories to each task (Work, Personal, Urgent, Health, Other)
* Delete tasks instantly
* Clean and intuitive input interface

---

### Mark Tasks as Complete

* Click the circular checkbox to mark a task as done
* Completed tasks are:

  * Visually dimmed
  * Striked through
* Click again to mark as incomplete

---

### Smart Filtering

Filter tasks based on:

* **All** – View everything
* **Active** – Pending tasks only
* **Done** – Completed tasks
* **Category-based filters** – Work, Personal, Urgent, Health

---

### Real-Time Stats

Track your productivity with:

* Total number of tasks
* Number of completed tasks
* Remaining tasks

---

### Progress Tracker

* Dynamic progress bar
* Percentage completion updates in real-time
* Visual motivation to complete tasks

---

### Aesthetic UI Design

* Soft peach, sunset, and rosy color palette
* Smooth animations and hover effects
* Minimal, distraction-free interface
* Responsive and centered card layout

---

### Dynamic Date Display

* Automatically shows the current day and date
* Updates every time the app is opened

---

### Clean State on Refresh

* App starts with an empty task list on every reload
* No stored or preloaded data (fresh start every time)

---

## Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling, layout, animations
* **JavaScript (Vanilla)** – Functionality and interactivity
* **Tabler Icons** – Iconography
* **Google Fonts** – Typography (Playfair Display, DM Sans)
